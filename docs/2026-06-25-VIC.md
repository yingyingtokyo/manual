## Data preparation for benchmark
### Meteorological data
We need to prepare 3-hour forcing data **air_temp, prec, pressure, swdown, lwdown, vp, wind** from data sources below:

- Daymet (https://search.earthdata.nasa.gov/search/granules?p=C2532426483-ORNL_CLOUD&pg[0][v]=f&pg[0][gsk]=-start_date&q=Daymet_Daily_V4R1_2129&ac=true&tl=552268799.999!5!!) <br>
  
  Daymet provides 1 km $\times$ 1 km daily forcings from 1980 to present.
  
    | Variable      |                Description (units)               |
    |:-------------:|:------------------------------------------------:|
    | tmax          | Daily maximum 2-meter air temperature (°C)       |
    | tmin          | Daily minimum 2-meter air temperature (°C)       |
    | prcp          | Daily total precipitation (mm/day)               |
    | swdown        | Incident shortwave radiation flux density (W/m2) |
    | vp            | Water vapor pressure (Pa)                        |

  ```bash
    #!/bin/bash
    base_url="https://data.ornldaac.earthdata.nasa.gov/protected/daymet/Daymet_Daily_V4R1/data"
    vars=("tmin" "tmax" "srad" "prcp" "vp")
    start_year=1980
    end_year=2024

    token="put_your_token"

    for var in "${vars[@]}"; do
      for year in $(seq $start_year $end_year); do
        filename="daymet_v4_daily_na_${var}_${year}.nc"
        url="${base_url}/${filename}"
        echo "Downloading: $filename"
        wget --auth-no-challenge -O "${filename}" "${url}"
        if [ $? -ne 0 ]; then
          echo "Failed to download $filename"
        fi
      done
    done
    ```

- Daymet-GSWP3 (https://doi.ccs.ornl.gov/dataset/48172fdb-751a-52ca-a456-9594f3739386) <br>
  Daymet-GSWP3 provides 3-hour time step historic weather forcing at 1-km spatial resolution. GSWP3 is used to fill the sub-daily temporal information.

- MetSim: A Python package for estimation and disaggregation of meteorological data (https://github.com/UW-Hydro/MetSim-tutorial) <br>
  MetSim can transform daily values into subdaily values.

- CONUS404 (https://gdex.ucar.edu/datasets/d559000/dataaccess/#) <br>
  
  CONUS404 provides 4 $\times$ 4 km daily forcings from 1980 to 2022. It is generated by dynamically downscaling through WRF from ERA5.
  (https://journals.ametsoc.org/view/journals/bams/104/8/BAMS-D-21-0326.1.xml)

  | Variable |          Description (units)              |
  |:--------:|:-----------------------------------------:|
  | u10      | U-component of wind at 10 meters (m/s)    |
  | v10      | V-component of wind at 10 meters (m/s)    |
  | PSFC     | Surface pressure (Pa)                     |
  | lwdown   | Downward longwave at ground surface (W/m2)|

 
  ```bash
  #!/bin/bash
  # Download and merge hourly wrf2d files into monthly NetCDF
  base="https://osdf-data.gdex.ucar.edu/ncar/gdex/d559000"
  for year in $(seq 2006 2010); do
    for month in $(seq -w 1 12); do
      if [ "$month" -ge 10 ]; then
        wy=$((year+1))
      else
        wy=$year
      fi
      # Days in this month
      ndays=$(cal $month $year | awk 'NF {DAYS = $NF}; END {print DAYS}')
      echo "Processing $year-$month (WY=$wy, $ndays days)"
      for day in $(seq -w 1 $ndays); do
        for hour in $(seq -w 0 23); do
          fname="wrf2d_d01_${year}-${month}-${day}_${hour}:00:00.nc"
          url="${base}/wy${wy}/${year}${month}/${fname}"
          if [ ! -f "$fname" ]; then
            #wget -q "$url" -O "$fname"
            wget -q -O "$fname" "$url" 2>/dev/null
          else
            echo "Already have $fname, skipping"
          fi
        done
      done
    done
  done
  ```

## Soil parameters
### CONUS-SOIL
CONUL-SOIL (http://dbwww.essc.psu.edu/dbndx/tree/amer_n/us_48/data/soilprop/statsgo_geo.html) provides 1km $\times$ 1km soil characteristics, such as texture, bulk density, and porosity, arranged in 11 standard layers. CONUS-SOIL layers 1 and 2 are converted into VIC model layer 1, CONUS-SOIL 3–7 to VIC model layer 2, and CONUS-SOIL layers 8 and 9 to VIC model layer 3.

  | Layer | Thickness | Depth to Top |   |
  |-------|-----------|--------------|---|
  | 1     | 5 cm      | 0 cm         |   |
  | 2     | 5 cm      | 5 cm         |   |
  | 3     | 10 cm     | 10 cm        |   |
  | 4     | 10 cm     | 20 cm        |   |
  | 5     | 10 cm     | 30 cm        |   |
  | 6     | 20 cm     | 40 cm        |   |
  | 7     | 20 cm     | 60 cm        |   |
  | 8     | 20 cm     | 80 cm        |   |
  | 9     | 50 cm     | 100 cm       |   |
  | 10    | 50 cm     | 150 cm       |   |
  | 11    | 50 cm     | 200 cm       |   |

- Soil texture <br>
STATSGO data have 16 texture types, including SAND, LOAMY SAND, SANDY LOAM, SILT LOAM, SILT, LOAM, SANDY CLAY LOAM, SILTY CLAY LOAM, CLAY LOAM, SANDY CLAY, SILTY CLAY, CLAY, ORGANIC MATERIAL, WATER, BEDROCK, OTHER (land-ice)

- Soil bulk density ($BD$)<br>
Soil bulk density derives from STATSGO database follows the providing the maximum (BDH) and minimum (BDL) values on a layer basis for each component in a STATSGO map unit. The mean value for BD for each STATSGO component layer was computed as,<br>
$BD =\frac{1}{2}[BDL + BDH]$.

- Soil porosity <br>
Porosity is a measure of the volume of air- and water-filled pores in the soil. A value for porosity can be calculated from the BD and the particle density (PD).<br>
$Porosity = 1 - [BD/PD]$

- $b_{infilt}$<br>
The b_infilt parameter is the parameter used to describe the Variable Infiltration Curve.<br>
$b_{infilt}=0.2$

- $D_s$<br>
The soil parameter $D_s$ represents the fraction of the Dsmax parameter at which non-linear baseflow occurs.<br>
$Ds = 0.001$

- $D_smax$<br>
The soil parameter $D_smax$ represents the fraction of the Dsmax parameter at which non-linear baseflow occurs. Slope is calculated from DEM.<br>
$D_smax = K_{sat} \times slope$

- $W_s$<br>
The parameter $W_s$ is the fraction of maximum soil moisture where non-linear baseflow occurs.<br>
$Ws = 0.9$

- $c$<br>
$c$ is exponent used in baseflow curve.<br>
$c=2$

- $expt$<br>
The $expt$ is the exponent ($n=3+2/\lambda$) from the Brooks-Corey relationship (https://vic.readthedocs.io/en/master/Documentation/soiltext/).

- $Ksat$<br>
Ksat is the saturated hydraulic conductivity. It can be assigned based on soil textures (https://vic.readthedocs.io/en/master/Documentation/soiltext/).

- $phi_s$<br>
Turn off (-999).

- $init_{moist}$<br>
$init_{moist} = porosity * layer depth * 1000$

- $elev$<br>
Average elevation of grid cell. 1/3 arcsec-resolution (∼ 10 m) USGS NED is used.

- $avg_T$<br>
Average annual air temperature.

- $dp$<br>
This is the soil thermal damping depth. It is defined as the depth in the soil column at which the soil temperature remains nearly constant annually. <br>
$dp=4$

- $bubble$<br>
The bubble parameter is the bubbling pressure. Only work in water and energy balanced mode. <br>
$bubble = 0.32*expt + 4.3$

- $quartz$<br>
Quartz content of soil (https://ral.ucar.edu/sites/default/files/public/product-tool/noah-multiparameterization-land-surface-model-noah-mp-lsm/soil_characteristics.html).

- $Soil_{density}$<br>
Soil particle density.<br>
$Soil_{density}=2685 kg/m^3$

- $off_{gmt}$	<br>
0 indicates that the model start date/time is relative to Greenwich Mean Time (GMT).

- $Wcr_{fract}$<br>	
Fractional soil moisture content at the critical point (~70% of field capacity)/(fraction of maximum moisture). From lookup table (https://extension.okstate.edu/fact-sheets/understanding-soil-water-content-and-thresholds-for-irrigation-management.html) (https://vic.readthedocs.io/en/master/Documentation/soiltext/).

- $Wpwp_{FRACT}$	<br>	
Fractional soil moisture content at the wilting point. From lookup table (https://vic.readthedocs.io/en/master/Documentation/soiltext/).

- $rough$	<br>	
Surface roughness of bare soil.<br>
$rough = 0.001$

- $snow_{rough}$ <br>	
Surface roughness of snowpack.<br>
$snow_{rough}=0.0005$

- $annual_{prec}$	<br>	
Average annual precipitation.

- $resid_{moist}$	<br>	
Soil moisture layer residual moisture content in units of residual moisture content volume / total layer volume [mm/mm].

- $fs_{active}$	<br>	
0, turn of frozen soil mode.

## Vegetation library
### Satellite datasets
#### MODIS-LAI (https://www.earthdata.nasa.gov/data/catalog/lpcloud-mcd15a2h-061)
- $LAI$
Monthly average of LAI from 2003 to 2008.

#### MODIS-albedo (https://www.earthdata.nasa.gov/data/catalog/lpcloud-mcd43a3-061)
- $albedo$<br>
Calculate average of albedo at the corresponding time of LAI data.

- $rarc$<br>
The parameter rarc represents the architectural resistance of each vegetation type. <br>
$rarc=2$.

- $rmin$<br>
The parameter rmin refers to the minimum stomatal resistance of each vegetation type.<br>
$rmin=100$.

#### LDAS (https://ldas.gsfc.nasa.gov/gldas/vegetation-parameters)
The LAI data are based on AVHRR satellite, 20-year climatology from 1982 - 2001.  The greenness fraction data are the Noah LSM default dataset from NCEP, based on AVHRR satellite, 5-year climatology from 1985-1989.

- $overstory$<br>
Indicate if the current vegetation type has an overstory or not.<br>

- $wind_h$<br>
Height at which wind speed is measured (10m). 

- $RGL$<br>
RGL is the minimum incoming shortwave radiation at which there will be transpiration. It typically has a value of 30 W/m2 for trees and for crops about 100 W/m2.

- $rad_atten$<br>
Raidation attenuation factor.

- $wind_atten$<br>
Wind speed attenuation through the overstory.

- $trunk_{ratio}$<br>
$trunk_{ratio}=0.2$
  
- $roughness length$<br>
Vegetation roughness length. $Z0MIN$ and $Z0MAX$ from LDAS paramater lib.

- $displacement$<br>
Vegetation displacement height
